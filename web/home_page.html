any method/employee?id={empId}Выводит работника с данным id
any method/employee/list Выводит псписок всех работников
post method/employee/createСоздает нового работника из информации
в теле запроса в формате json
any method/employee/remove?id={empId}Удаляет существующего работника с данным id
post method/employee/updateИзменяет информацию о существующем работнике
на информацию в теле запроса в формате json

Пример json:
{
"Id":1,(требуется только при изменении работника)
"Name":"SomeName",
"Surname":"SomeSurname",
"Phone":"+12345678910",
"CompanyId":1,
"EmpPassport":{
"Type":"Passport RF",
"Number":"1246785317"
},
"EmpDepartment":{
"Name":"Administration"(Выбор осуществляется только из существующих)

}

Существующие департаменты:
IT
Sales
Administration
Accounting
